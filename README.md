# 微信小程序
一个基于wepy框架开发的小程序

## 参考文档
- [wepy：组件化的小程序开发框架](https://github.com/wepyjs/wepy)
- [小程序开发文档](https://mp.weixin.qq.com/debug/wxadoc/dev/index.html)

## 下载
```
git clone https://github.com/kolf/wxApp.git
```

## 开始使用
全局安装wepy
```
npm install -g wepy
```
进入项目安装依赖包
```
npm install
```
开发
```
wepy build --watch
```
